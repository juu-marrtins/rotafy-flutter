Rotafy

Aplicativo de caronas entre universidades

Rotafy é uma plataforma criada para conectar estudantes de diferentes universidades que desejam compartilhar caronas de forma prática, segura e econômica. O aplicativo oferece perfis para passageiros e motoristas, permitindo solicitações rápidas e uma gestão simples das viagens.

Visão Geral

O objetivo do Rotafy é facilitar deslocamentos acadêmicos, reduzindo custos, otimizando rotas e promovendo sustentabilidade.
Atualmente, o MVP inclui:

Fluxo de registro e login.

Home do passageiro com filtro de rotas e solicitação de corrida.

Home do motorista com visualização de corridas recentes.

Estrutura pensada para escalabilidade futura (pagamentos, chat, verificação universitária etc.).

Funcionalidades Principais
1. Registro e Autenticação

Criação de conta com dados básicos.

Login seguro.

Armazenamento de informações do usuário (passageiro ou motorista).

2. Perfil do Passageiro

Tela inicial com filtros por origem, destino e horário.

Solicitação de corrida diretamente pelo aplicativo.

Visualização dos motoristas disponíveis (versão futura).

3. Perfil do Motorista

Exibição das corridas mais recentes solicitadas.

Acesso rápido às informações dos passageiros.

Possibilidade de aceitar ou recusar solicitações (versão futura).

Arquitetura do Projeto

O aplicativo está estruturado para permitir rápida expansão:

Frontend:
Desenvolvido para proporcionar uma interface intuitiva e responsiva.

Backend:
API preparada para autenticação, gestão de usuários e operações relacionadas às corridas.

Banco de Dados:
Estruturado para armazenar usuários, rotas, solicitações e histórico de corridas.
